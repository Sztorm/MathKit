RegularPolygon

interface RegularPolygon : RegularShape, Transformable

Represents a transformable regular polygon in a two-dimensional Euclidean space.

Implementations that use default-implemented members of this interface must make sure that the properties center, orientation, sideLength, sideCount and the copy method are independent of other members and the computational complexity of these members is trivial.
